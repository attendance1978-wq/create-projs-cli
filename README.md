## Installation

### Run directly with `npx` (no installation needed)
```bash
npx attendance1978-wq/create-projs-cli
```

### Optional: Install globally for shorter commands
```bash
npm install -g attendance1978-wq/create-projs-cli
```

### After global install, you can run:

```bash
attendance my-project
```
